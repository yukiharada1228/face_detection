# face_detection
 OpenVINOによる顔検出

# DEMO
!(img/demo1.png)
 
# Features
 近く写った顔から順に，指定した数まで検出します．
 
# Requirement
* openvino 2021.4.2
* openvino-dev[pytorch] 2021.4.2
 
# Installation
```bash
pip install -r requirements.txt
```

# Usage
```bash
git clone https://github.com/yukiharada1228/face_detection.git
cd face_detection
python face_detect.py
```
